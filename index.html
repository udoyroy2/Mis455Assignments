Assignment 1

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the basic structure of a website by organizing content such as headings, paragraphs, images, links, tables, and forms. HTML uses tags and elements to define different parts of a webpage, allowing browsers to display content correctly. Every website on the internet relies on HTML as its foundation.

HTML Essential HTML tags that every developer should remember include h1 to h6 for defining headings, p for writing paragraphs, a for creating hyperlinks, img for displaying images, ul, ol, and li for creating unordered and ordered lists, table, tr, th, and td for organizing data in tabular form, and form, input, label, and button for collecting user input. Semantic tags such as header for the top section of a webpage, nav for navigation links, main for the primary content, section for grouping related content, article for independent content, and footer for the bottom section improve code readability and accessibility. Important attributes like id (unique identifier), class (grouping elements for styling), href (link destination), src (resource location), and alt (alternative text for images) should also be remembered, as they are frequently used in web development.
